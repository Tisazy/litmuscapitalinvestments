# Litmus Capital Investments Website

This repository contains the source code for the official website of Litmus Capital Investments.

## Project Structure

*   `index.html`: The main HTML file.
*   `src/css/style.css`: The stylesheet for the website.
*   `src/js/script.js`: The JavaScript file for the website.

## Deployment

This website is deployed on Vercel and is accessible at [litmuscapitalinvestments.co.za](https://litmuscapitalinvestments.co.za).

Changes pushed to the `main` branch of this GitHub repository will be automatically deployed to the live website.
